# Presupuesto Familiar Bot

Bot de Telegram para control de presupuesto personal y familiar.

## Tecnologías

- Python
- FastAPI
- PostgreSQL (Supabase)
- Telegram Bot API
- Arquitectura basada en servicios y API

## Funcionalidades

- Registro de gastos
- Registro de ingresos
- Resumen mensual
- Consulta de gastos
- Paginación de movimientos
- Categorías aprendidas por usuario
- Configuración de moneda y fecha de corte
- Exportación de gastos

## Ejecución

API:

```bash
python run_api.py
```

Bot:

```bash
python main.py
```
